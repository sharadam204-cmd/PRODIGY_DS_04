PRODIGY_DS_04


This project focuses on analyzing and visualizing sentiment patterns in social media data to understand public opinion towards different topics and brands. The analysis was performed using Python in Google Colab with data visualization techniques to identify positive, negative, neutral, and irrelevant sentiments.





Features:

Data preprocessing and cleaning

Sentiment distribution analysis

Bar chart and pie chart visualizations

Comparison between training and validation datasets

Public opinion analysis on various topics and brands






Technologies Used:

Python

Pandas

Matplotlib

Google Colab



Dataset: Twitter sentiment analysis dataset containing tweets categorized into different sentiment classes such as Positive, Negative, Neutral, and Irrelevant.



Outcome: Successfully analyzed social media sentiment patterns and visualized insights to better understand user opinions and engagement trends.
